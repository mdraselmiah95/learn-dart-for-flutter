# Lesson 03 — Operators

## Topics Covered
- Arithmetic: `+` `-` `*` `/` `~/` `%`
- Assignment: `=` `+=` `-=` `*=` `/=` `++` `--`
- Comparison: `==` `!=` `>` `<` `>=` `<=`
- Logical: `&&` `||` `!`
- Null-aware: `??` `??=` `?.`
- Type test: `is` `is!` `as`

## Flutter Connection
Operators are used everywhere in Flutter — calculating layouts, comparing state values, and null-safe property access with `?.`.

## How to Run
```bash
dart run lessons/lesson-03-operators/main.dart
```
