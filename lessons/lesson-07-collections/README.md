# Lesson 07 — Collections

## Topics Covered
- `List` — create, access, add, remove, sort, loop
- `Map` — create, access, add, remove, loop keys/values
- `Set` — unique values, add, contains, intersection
- Spread operator `...`
- Collection methods: `map`, `where`, `reduce`, `any`, `every`, `firstWhere`

## Flutter Connection
Flutter apps are full of collections — a list of users, a map of settings, a set of selected items. `ListView.builder` renders a `List` directly. Understanding collections is non-negotiable for Flutter.

## How to Run
```bash
dart run lessons/lesson-07-collections/main.dart
```
