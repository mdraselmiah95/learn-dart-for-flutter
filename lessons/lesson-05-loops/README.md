# Lesson 05 — Loops

## Topics Covered
- `for` loop with counter
- `for-in` loop through collections
- `while` loop
- `do-while` loop
- `break` and `continue`
- `forEach` on List and Map

## Flutter Connection
Loops power dynamic UI in Flutter — rendering a list of widgets from a `List` uses `for` or `.map()`, both learned here.

## How to Run
```bash
dart run lessons/lesson-05-loops/main.dart
```
