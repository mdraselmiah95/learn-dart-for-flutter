# Lesson 04 — Control Flow

## Topics Covered
- `if` / `else if` / `else`
- `switch` / `case` / `default` / `break`
- Ternary operator: `condition ? a : b`
- Nested conditions

## Flutter Connection
Control flow drives UI logic in Flutter — showing/hiding widgets, checking auth state, handling empty lists vs filled lists.

## How to Run
```bash
dart run lessons/lesson-04-control-flow/main.dart
```
