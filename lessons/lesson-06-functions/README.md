# Lesson 06 — Functions

## Topics Covered
- Positional parameters
- Named parameters (`{required}`)
- Optional parameters with defaults
- Arrow functions (`=>`)
- Anonymous functions
- Higher-order functions (passing functions as arguments)
- `map`, `where`, `reduce`, `sort`

## Flutter Connection
Flutter widgets are built and rebuilt by calling functions. Callbacks like `onPressed: () {}` are anonymous functions. Understanding higher-order functions makes Flutter's `.map()` widget builder pattern easy.

## How to Run
```bash
dart run lessons/lesson-06-functions/main.dart
```
