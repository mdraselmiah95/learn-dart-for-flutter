# Lesson 08 — Null Safety

## Topics Covered
- Why null safety exists
- Nullable types with `?`
- Null check with `if (x != null)`
- Null-aware access with `?.`
- Null-coalescing with `??`
- Null-coalescing assignment `??=`
- Force unwrap with `!` (and when NOT to use it)
- `late` variables
- Null safety with collections

## Flutter Connection
All modern Flutter apps (Flutter 2.0+) are null-safe. You will see `?`, `??`, and `?.` in every Flutter codebase. Without understanding null safety you will not be able to read or write Flutter code.

## How to Run
```bash
dart run lessons/lesson-08-null-safety/main.dart
```
