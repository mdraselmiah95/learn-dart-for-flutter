# Lesson 09 — OOP: Classes & Objects

## Topics Covered
- What is a class and an object
- Properties (fields) and methods
- Default constructor
- Named constructors
- `this` keyword
- Getters and setters
- Private fields with `_`
- `toString()` override
- Factory constructors

## Flutter Connection
Every Flutter widget is a class. `StatelessWidget`, `StatefulWidget`, your data models — all classes. Understanding OOP is the single most important step before starting Flutter.

## How to Run
```bash
dart run lessons/lesson-09-oop-classes/main.dart
```
