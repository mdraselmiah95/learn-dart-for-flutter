## Why Learn Dart Before Flutter?

Flutter is 100% built on Dart. Without Dart:

- You will copy-paste Flutter code without understanding it
- Debugging will feel impossible
- State management, async calls, and models will confuse you

**Learn Dart first → Flutter will make complete sense.**

---

## What You Will Learn

- Dart syntax, variables, and data types
- Operators, conditions, and loops
- Functions — positional, named, arrow, anonymous
- Collections — List, Map, Set
- Null safety — Dart's most important feature
- Object-Oriented Programming — class, constructor, inheritance
- Async programming — Future, async/await, Streams
- Real code patterns used in every Flutter app

## How to Run Any Lesson

```bash
# Run a lesson
dart run lessons/lesson-01-introduction/main.dart

# Run your practice file
dart run lessons/lesson-01-introduction/practice.dart

# Check the solution
dart run lessons/lesson-01-introduction/solution.dart
```

> No setup needed for quick testing — use [DartPad](https://dartpad.dev) in your browser.
