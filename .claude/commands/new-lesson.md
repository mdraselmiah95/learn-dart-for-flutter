# /project:new-lesson

Create a new Dart lesson file for a given topic.

## Usage
/project:new-lesson 3 operators

## Steps
1. Create folder `lesson-$ARG1/`
2. Create `lesson-$ARG1/lesson-$ARG1.dart`
3. Fill with topic ($ARG2) examples with beginner-friendly inline comments
4. Update CLAUDE.md lesson structure list
